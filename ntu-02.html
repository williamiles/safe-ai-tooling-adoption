Article 2

Something will go wrong.
Here's how to limit the damage.

You can't prevent every mistake. But you can make sure one mistake doesn't ruin everything.

The mindset shift

One shift changes everything

Stop trying to be perfect. Start building walls between your mistakes.

😰
The old thinking

"I need to never make a mistake."

Stressful. Impossible. One slip and everything falls.

🧹
The security mindset

"What could go wrong? And how can we minimize the consequences!"

Calm. Realistic. Damage stays small.

🔑

You already know this pattern

You use a different password for every account. Why? Because if one gets stolen, the thief can't get into everything else. That's containment. The same idea works for AI tools: don't let one mistake give away everything.

What happens when...

Tap each scenario to see the risk, and the safer alternative.

💬 You paste your whole inbox into the AI for help? +
Risk: The AI now has every name, address, password reset link, and private conversation. One trick and all of it can leak.
Better: Copy only the specific email you need help with. Leave the rest out.
📂 You give the AI access to your entire drive? +
Risk: Tax returns, contracts, personal photos. The AI can read all of it. A hidden instruction in one file could expose everything else.
Better: Share only the specific folder or file you're working with. Remove access when you're done.
✅ You say "yes" to every permission the AI asks for? +
Risk: The AI can now send emails, read your calendar, and browse the web on your behalf. A single trick could use all of those powers at once.
Better: Read each permission request. Ask: "Does the AI actually need this for what I'm doing right now?" If not, say no.
🔄 You trust the AI's output without checking? +
Risk: The AI might have been tricked and is now showing you wrong information, or quietly doing something you didn't ask for.
Better: Glance at what the AI did before you send, save, or share. If something looks odd, like a link you didn't expect or a message you didn't write, pause and check.

Five containment habits

The security mindset in five daily habits. Tap each one you'll try.

Share the minimum, only the file or text the AI actually needs
Read permission requests before tapping "allow"
Keep sensitive stuff separate. Don't mix personal files with work AI
Glance before you send. Check the AI's output before sharing it with others
Revoke access you no longer need. Treat permissions like borrowed keys
